## Projeto de Pós-Doutorado

### Low-complexity Approximations of the WECS Method for SAR Change Detection

**Autores:**  
Luan Portella, Renato J. Cintra, Aluísio Pinheiro

### Resumo

Este projeto de pós-doutoramento investiga novas abordagens de baixa complexidade para o método Wavelet Energy Correlation Screening (WECS), empregado na detecção de mudanças em séries multitemporais de imagens SAR. O WECS utiliza a transformada wavelet não dizimada (ND-DWT) para extrair coeficientes de aproximação que auxiliam na identificação de alterações ao longo do tempo. Embora eficiente, o método apresenta alto custo computacional devido ao número de filtragens wavelet necessárias sem subsampling.

Neste trabalho, desenvolvemos aproximações eficientes para os filtros utilizados no WECS, baseadas em versões truncadas da representação Canonical Signed Digit (CSD). Essas aproximações reduzem de forma significativa a quantidade de operações aritméticas necessárias para gerar os coeficientes wavelet, mantendo, na prática, o desempenho do método original. Resultados obtidos com dados simulados e reais indicam que as versões aproximadas não apenas preservam a acurácia do WECS tradicional, como também podem superá-lo em determinados cenários, oferecendo uma alternativa mais eficiente para aplicações de sensoriamento remoto e análise de séries temporais SAR em larga escala.
